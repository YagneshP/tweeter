## Getting Started

### Prerequisites

* node
* Express

### Installation

1. Fork this repo
2. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the web server. The app will be served at <http://localhost:8080/>
   ```sh
   npm run local
   ```
5. Go to <http://localhost:8080/> in your browser.
